## Struktur Knowledge Files

Setiap knowledge file mengikuti standar struktur:

1. **Overview** - Ringkasan umum dan statistik utama
2. **Tabel Utama** - Data lengkap dalam format HTML table
3. **Tabel Ringkasan** - Summary per kategori/industri/kuartal
4. **Analisis** - Analisis root cause & effect, pola temporal, modus dominan
5. **Rekomendasi** - Saran kebijakan berbasis data
6. **Metodologi** - Catatan tentang verifikasi dan sumber data
